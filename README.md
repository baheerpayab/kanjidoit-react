# Kanji Do It

An app to help you learn Kanji on your journey to master Japanese.

In order to practice CSS skills, without wasting time pondering on design, the app attempts to imitate the Fiori Horizon design system and UI5 components.
A custom design is to follow once a minimum-viable-product is fulfilled.

This is a personal project aimed at developing React, Typescript, and general web-engineering skills.
